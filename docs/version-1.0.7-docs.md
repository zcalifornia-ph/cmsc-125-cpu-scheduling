## Pitfalls and Debugging Guidance

### 1) Stale screenshots after source edits

Any source change that affects user-visible output requires a screenshot refresh. Check `artifacts/screenshots/` after every source polish pass.

### 2) Cropping inconsistency

Ensure screenshots are cropped consistently across all four captures so the report layout looks uniform.

### 3) texput.log not committed

The LaTeX auxiliary file `cmsc-125-cpu-scheduling/writeup/texput.log` is a local build artifact and is not tracked. Do not add it to the repository.
