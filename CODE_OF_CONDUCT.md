# Code of Conduct

## Our Commitment

This repository aims to maintain a respectful, focused, and evidence-based collaboration environment.
All participation in issues, pull requests, code review, documentation, and discussion is expected to support:

- respectful communication
- academic integrity
- accurate technical claims
- constructive feedback
- responsible handling of code, results, and citations

## Expected Behavior

- Communicate clearly and professionally.
- Critique code, logic, and evidence rather than people.
- Keep technical claims reproducible and honest.
- Credit sources when external material is adapted or reused.
- Preserve a learning-oriented environment when discussing algorithms, metrics, and implementation tradeoffs.

## Unacceptable Behavior

- Harassment, intimidation, or personal attacks.
- Discriminatory or hateful language.
- Plagiarism, misrepresentation of authorship, or fabricated results.
- Knowingly submitting misleading benchmarks, screenshots, or comparison data.
- Sharing private information without consent.

## Scope

This Code of Conduct applies to all project spaces associated with this repository, including:

- issues
- pull requests
- commit discussions
- documentation updates
- direct collaboration tied to this project

## Reporting

Report concerns to <zecalifornia@up.edu.ph>.
Include links, screenshots, logs, or other context when relevant.
Reports will be handled as discreetly as practical.

## Enforcement

Project maintainers may remove content, restrict participation, or reject contributions that violate this Code of Conduct.
Enforcement decisions should prioritize fairness, safety, and the integrity of the project.
